初始化 GitHub 同步
博客默认将文章保存到 GitHub 仓库。请按照以下步骤配置：
1

创建 Personal Access Token

访问 GitHub Settings，创建 Classic Token，勾选 repo 权限

前往创建 Token
2

填写仓库信息

输入你的 GitHub 用户名、仓库名（如 username.github.io）和 Token

3

测试并保存

点击"测试连接"验证配置，然后保存即可开始写作

数据同步管理
文章会自动同步到 GitHub 仓库的 data/posts.json 文件中。这是推荐的存储方式，数据安全可靠。

GitHub 配置

需要在 GitHub Settings > Developer settings > Personal access tokens 中创建，勾选 repo 权限
警告： 本地存储模式仅将数据保存在浏览器中。清空浏览器数据会导致文章丢失，建议仅用于临时测试。
